### Installation

   * First, install [packer.nvim](https://github.com/wbthomason/packer.nvim).
   * Then, clone this repo 
   ```
     $ git clone https://github.com/janleigh/neodots $HOME/.config/nvim
   ```
   * Run `:PackerSync` and restart nvim.
